# 项目结构
|- module name----------------------模块名
    |- controllers----------------------控制器文件夹
        |- all controller file--------------详细的控制器文件
    |- models---------------------------模型文件夹
        |- all model file-------------------控制器通过模型文件操作收据，模型文件
                                                引用db.models
    |- views----------------------------视图文件夹
        |- all view file--------------------ui文件添加槽方法
        |- all ui.py file-------------------ui文件转换成的视图文件
    |- modules-------------------------------信号和槽的连接文件
        |- all module file----------------------信号和槽的连接文件，继承ui文件
    |- images---------------------------图片文件夹
        |- all images-----------------------所有的图片
        |- qrc file-------------------------qtdesigner生成的资源文件
        |- rc file--------------------------qrc文件转为py文件
|- db-------------------------------数据库文件夹
    |- models---------------------------存放Django的所有models,其他模块连接
                                            数据库引用该模块
|- lib------------------------------共用模型资源
    |- all model file-------------------所有的模型文件
|- images---------------------------共用图片资源
    |- all images-----------------------所有的图片
